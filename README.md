# Tikloop
The Future of Short Videos on Web3


TikLoop adalah platform video Web3 berbasis Base Network yang menghadirkan pengalaman short video mirip TikTok, namun dengan kepemilikan data, reward berbasis on-chain, dan ekonomi kreator yang lebih adil. Pengguna dapat membuat, menonton, dan berinteraksi dengan konten sambil mendapatkan reward dari aktivitas mereka. TikLoop membuka peluang baru bagi kreator untuk memonetisasi karya mereka tanpa perantara, transparan, dan sepenuhnya on-chain.
